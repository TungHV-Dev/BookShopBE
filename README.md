# BookShopBE
1. Overview
- This is a web api project which is built base on ASP.NET Core 3.1
- Technology:
+ ASP.NET Core 3.1
+ Entity Framework Core
+ Dapper
+ Dependency Injection
+ Auto Mapper

2. Project structure
- The project structure consists of 5 part:
+ Common
+ Model
+ Data
+ Service
+ Web API
